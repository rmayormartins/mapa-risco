Sobre o Cruzamento Espacial

Metodologia:
1. Dados CEMADEN fornecem localização geográfica de áreas monitoradas
2. API do IBGE retorna população de Itajaí: 264.054 pessoas (Censo 2022)
3. Análise espacial cria buffer (área de influência) ao redor de cada ponto
4. População é distribuída considerando densidade urbana e nível de risco
5. Perfil demográfico de Itajaí: 19% idosos, 21% crianças, 6.7% PcD
6. Sistema calcula população vulnerável em cada área de risco
