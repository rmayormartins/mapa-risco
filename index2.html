Como usar este sistema

1. Baixe os dados do CEMADEN em mapainterativo.cemaden.gov.br
2. Carregue o arquivo CSV usando o botão acima
3. Clique em Buscar Dados IBGE para enriquecer com dados demográficos da API real
4. Use os filtros para analisar áreas específicas
5. Exporte os resultados em CSV
